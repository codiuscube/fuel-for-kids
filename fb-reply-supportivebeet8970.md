# FB reply to SupportiveBeet8970

Yeah you're tracking right. SB 2 caps the program administrator's fee at 5%, so $50M is the ceiling — that's the basis for the "Odyssey takes $50M" reports. The appeals reserve is real (the comptroller's April 28 update mentions it as item 5) but the dollar amount hasn't been published yet, so $5M is a reasonable guess.

If we run with $1B − $415M (T1 + siblings) − $50M (Odyssey) − $5M (reserve) = ~$530M for Tier 2, that's about 62,000 T2 slots at the blended private/homeschool rate (~$8,525/student). So T2 would fund at ~85% instead of the ~94% you get assuming the full $585M.

For Tier 3 families like mine, the cascade math at 15% attrition shifts from ~48% family-of-3 down to ~27% per family. Still meaningful but materially lower. Worth keeping in mind as a downside scenario until the comptroller publishes actual admin and reserve figures.
